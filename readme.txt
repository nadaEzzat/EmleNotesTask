- Chat Application using java programming language.

- Pakages are :
	- main : for the main page.
	- chat : secend page in the app.
	- model
	- adapter : conatins all adapter classes that used in the app.
	- network : fetching data from the URLs using Retrofit and
		    check internet conncection.

Code Features :
	- Build with MVP "Model-View-Presenter" Design Pattern.

	- API requests made with Retrofit.

	- Binding views using butterknife.

	- This app Check internet connection before fetching the data from the URLs.

	- you can send message in chat page.

- Libraries : 
	- Retrofit
	- Butterknife
	- Recyclerview
	- Picasso
	- CircleImageView
